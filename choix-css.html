Choix CSS – Projet Kit-Kat Big Forms

Lien avec le cours du 09.03 – Découverte CSS

Choix CSS 1 — Box Model

Notion utilisée :

J'ai utilisé le padding et le border-radius.

Où dans mon code :

Dans la classe .input-field

Extrait :

.input-field {
  padding: 0.625rem 1rem;
  border-radius: 0.5rem;
  border: 1.5px solid #ddd;
}

Effet sur le rendu :

Le texte ne touche plus les bords du champ et les angles arrondis donnent un style plus doux et moderne aux zones de saisie.

Choix CSS 2 — Flexbox

Notion utilisée :

J'ai utilisé display: flex avec justify-content et align-items.

Où dans mon code :

Dans la classe .card-header

Extrait :

.card-header {
  display: flex;
  align-items: center;
  justify-content: space-between;
  gap: 1rem;
}

Effet sur le rendu :

Le logo et le titre restent alignés et bien espacés sur la même ligne, peu importe la taille de l'écran.

Choix CSS 3 — Transitions et effet au survol (:hover)

Notion utilisée :

J'ai utilisé la pseudo-classe :hover combinée avec transition.

Où dans mon code :

Dans la classe .submit-btn

Extrait :

.submit-btn {
  background: #d4000e;
  transition: background 0.15s;
}

.submit-btn:hover {
  background: #b5000c;
}

Effet sur le rendu :

Le bouton change de couleur en douceur quand la souris passe dessus, ce qui rend l'interface plus interactive et agréable à utiliser.

Choix CSS 4 — Police personnalisée (@font-face)

Notion utilisée :

J'ai utilisé @font-face pour importer une police personnalisée.

Où dans mon code :

Dans .header-title

Extrait :

@font-face {
  font-family: "KittyKatt";
  src: url("fonts/KittyKatt.ttf") format("truetype");
}

.header-title {
  font-family: "KittyKatt", Georgia, serif;
  font-style: italic;
  letter-spacing: 2px;
}

Effet sur le rendu :

Le titre du formulaire a une police originale qui correspond au thème Kit-Kat, au lieu d'utiliser une police par défaut.

Choix CSS 5 — Positionnement (position: absolute)

Notion utilisée :

J'ai utilisé position: absolute avec z-index pour placer des décorations.

Où dans mon code :

Dans les classes .shape, .shape-left et .shape-right

Extrait :

.shape {
  position: absolute;
  bottom: 0;
  z-index: 1;
}

.shape-left {
  left: 0;
  width: 22%;
}

Effet sur le rendu :

Les illustrations restent fixées dans les coins inférieurs de la page, derrière le formulaire, sans déranger l'affichage des champs.
